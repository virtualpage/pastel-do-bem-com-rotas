# Pastel do Bem

**Pastel do Bem** é uma iniciativa do **Instituto Alpha do Bem**, destinada à venda de *pastéis de Belém*, com **100% da renda revertida para projetos sociais** da ONG.  
Este repositório contém o código-fonte da aplicação web do projeto.

---

## Tecnologias Utilizadas

Este projeto foi desenvolvido com:

- **Vite**
- **React**
- **TypeScript**
- **TailwindCSS**
- **React Icons**
